Description

Flujo Normal

  1. Funcionario ingresa a sistema SAU
  2. Funcionario digita cédula / RUC del usuario
  3. Funcionario contrasta información física con formulario impreso del usuario
  4. Funcionario informa al sistema SAU que la información del usuario es correcta y real
  5. Funcionario procede a generar archivo XML con ficha única
  6. Sistema SAU firma electrónicamente el documento generado.
  7. Funcionario notifica al usuario que el proceso se ha completado exitosamente.

Flujo Alterno

  1. Funcionario ingresa a sistema SAU
  2. Funcionario digita cédula / RUC del usuario
  3. Funcionario contrasta información física con formulario impreso del usuario
  4. Funcionario informa al sistema SAU que la información del usuario es incorrecta.
  5. Sistema SAU emitira una alerta notificando posible adulteracion de identidad del usuario
  6. El canal autorizado ejecutará acciones de verificación de acuerdo a sus políticas internas establecidas para estos casos
