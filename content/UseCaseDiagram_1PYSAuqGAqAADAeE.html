  Cambio de contraseña : Use Case

Flujo Normal Cambio de Contraseña

  1. Usuario solicita camnio de contraseña
  2. SAU presenta opción de cambio de contraseña
  3. Usuario ingresa nueva contraseña
  4. Sistema valida que la contraseña cumpla reglas de clave
  5. Sistema notifica que cambio de contraseña es correcto

Flujo Alterno Cambio de Contraseña

  1. Usuario solicita camnio de contraseña
  2. SAU presenta opción de cambio de contraseña
  3. Usuario ingresa nueva contraseña
  4. Sistema indica que la contraseña no cumple con las normas
  5. Usuario ingresa nueva contraseña
  6. Sistema valida contraseña (Se repite bucle 4-6 hasta que la contraseña sea correcta)
  7. Sistema notifica que cambio de contraseña es correcto
  Desbloqueo de cuenta : Use Case

Flujo Normal

  1. Usuario ingresa a Portal SAU
  2. SAU valida que la cuenta está bloquedad
  3. SAU muestra mensaje de cuenta bloqueada
  4. SAU envía notificación a email de usuario
  5. Usuario recibe notificación de bloque con indicaciones para desbloqueo
  6. SAU valida si debe desbloquear cuenta de manera automática
  7. Usuario espera el tiempo establecido
  8. Usuario accede al sistema SAU

  4. Sistema valia que la contraseña sea válida
  5. Sistema notifica que cambio de contraseña es correcto
  Recuperación de contraseña : Use Case

Flujo Normal

  1. Usuario ingresa a Portal SAU
  2. Usuario accee a la opción "Recuperar Contraseña"
  3. SAU valida políticas de contraseña
  4. Si se cumplen las condiciones e contraseña
  5. SAU envía OTP a correo electrónico y SMS
  6. Usuario ingresa OTP
  7. SAU valida que no se trate de cuenta bloqueada
  8. Usuario recibe notificación de cambio de contraseña
  9. Usuario ingresa nueva contraseña      
  Suspensión manual de cuenta : Use Case

Flujo Normal

  1. Usuario ingresa al portal SAU
  2. Usuario accede a la administración
  3. Usuario selecciona "Suspender Cuenta"
  4. Sistema SAU advierte al usuario que se va a suspender la cuenta
  5. Usuario confirma decisión de suspender cuenta
  6. Sistema SAU envía OTP a correo electrónico y SMS
  7. Usuario ingresa OT
  8. Sistema realiza suspensión de cuenta
  9. Usuario recibe notificación de cuenta suspendida
